Experiments:

Experiments can be executed by running Exp1.
The arguments are as follows: 
1. lin/KD for linear or KD rangequery,
2. EPS (distance)
3. CSV filename
4. X coordinate
5. Y cooridnate
6. Z coordinate
7. Experiment number (exp1, exp2)

Running experiment 1 will give the nearest neighbors. It will also give the Experiment 2 time result following the list of neighbors.
